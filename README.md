# P2HACKS2025 アピールシート

## プロダクト名
U'n'IVERSE(ユニバース)

## コンセプト
対面でのコミュニケーションを円滑にする、学生向けの気軽な名刺交換サービス

## 対象ユーザ
日ごろから創作活動に励み人間関係を広げたいと考える学生。
様々な分野の創作に興味を持ち、同じ趣味を持つ人を探している。
気が合う友達を作りたいがすでにコミュニティができており、もうだれかに話しかける雰囲気ではないなと思っている。

## 利用の流れ
スキャンを選択するとカメラモードに切り替わり、相手の名刺のQRコードを読み取ることができます。また、サインアップして自分の名前、肩書、ソーシャルリンクを入力すると、自分の名刺を作成することができ、相手との名刺交換をすることができるようになります。名刺を交換すると、画面下には交換した相手とのつながりを表すユニバースが展開されます。

## 推しポイント
一般的な堅い雰囲気の名刺交換を、気さくな挨拶を交わすようなカジュアルなものへと変化させた本プロダクトは名刺一枚で相手のソーシャルアカウントへのアクセスを簡単に行うことができます。宇宙をテーマにした本プロダクトは、全体的な配色から落ち着いたテーマを見せながら名刺のデザインやプロダクトロゴから未来のテクノロジー感を見せてくれます。本プロダクトの重要機能「UNIVERSE」では今までに名刺を交換した人の交友関係を光の線でつなぎ、まるで星座のような関係図が表れます。

### 開発における工夫した点
プロジェクトを進めるにあたって、円滑なコミュニケーションを重要視し、対面でのコミュニケーションにおける工夫点としては、チームメンバー全員で昭和温泉に行き入浴、食事、会議を通して親睦を深めました。また、オンラインでのコミュニケーションにおける工夫点としては、Discordで通話をしながら作業を進めたり、メンバー各々のtimesチャンネルを設け、そこに各自報告等を書き込むなどがありました。さらに、スムーズな開発を実現すべく、作業前には情報共有を行いチーム全体の共通認識を作りました。そこでは、プロダクトの方向性であったり、メンバーの希望する機能であったり、プロダクトの魅力について話し合いました。また、事前に技術スタックを決め、練習プロダクトを制作しました。
